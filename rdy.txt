  3 Niveaux (L1/L2/L3)

  2. L2 - Short-term Memory (Court terme)
    - Session actuelle + fichiers récents

  3. L3 - Long-term Memory (Long terme)
    - Stockée dans PostgreSQL (table memories)
    - Patterns architecturaux, décisions, apprentissages
    - Persistent entre les sessions

  Comment la Mémoire se Renforce

  Le renforcement se fait via access_count dans la table system_rules de PostgreSQL :

  -- Chaque fois qu'une règle est utilisée :
  UPDATE system_rules
  SET access_count = access_count + 1,
      strength = LEAST(1.0, strength + 0.1)
  WHERE rule_id = ...

  Qui Renforce la Mémoire ?

  C'est automatique via le tool think() du MCP agi-tools :
  - Chaque fois que je think("quelque chose"), le système :
    a. Query les règles pertinentes
    b. Incrémente leur access_count
    c. Augmente leur strength
    d. Les règles les plus utilisées deviennent prioritaires

  # /home/pilote/projet/agi/cortex/l1_observer_daemon.py

  """
  Agent qui observe TOUTES les conversations Claude
  Via monitoring du fichier history.jsonl
  Stocke automatiquement SANS intervention Claude
  """

  import asyncio
  import json
  import asyncpg
  from pathlib import Path
  from watchdog.observers import Observer
  from watchdog.events import FileSystemEventHandler

  HISTORY_FILE = Path.home() / ".claude" / "history.jsonl"

  class ConversationObserver(FileSystemEventHandler):
      def __init__(self, db_pool):
          self.pool = db_pool
          self.last_position = 0

      def on_modified(self, event):
          if event.src_path == str(HISTORY_FILE):
              asyncio.create_task(self.process_new_messages())

      async def process_new_messages(self):
          """Lit nouvelles lignes dans history.jsonl"""

          with open(HISTORY_FILE, 'r') as f:
              f.seek(self.last_position)
              new_lines = f.readlines()
              self.last_position = f.tell()

          for line in new_lines:
              msg = json.loads(line)

              # Détecte apprentissage important
              if self.is_important(msg):
                  await self.store_automatically(msg)

      def is_important(self, msg):
          """Score importance 0-1"""
          keywords = ["architecture", "decision", "pattern", "bug fix", "implementation"]
          text = msg.get('content', '').lower()

          score = sum(1 for kw in keywords if kw in text) / len(keywords)
          return score > 0.3

      async def store_automatically(self, msg):
          """Stocke SANS demander à Claude"""

          async with self.pool.acquire() as conn:
              await conn.execute("""
                  INSERT INTO agi_knowledge (section, content, tags, priority)
                  VALUES ($1, $2, $3, $4)
              """,
                  "auto-learned",
                  msg['content'],
                  ["auto-stored", "conversation"],
                  5
              )

          print(f"💾 AUTO-STORED: {msg['content'][:100]}...")

  async def main():
      pool = await asyncpg.create_pool("postgresql://agi_user:agi_password@localhost/agi_db")

      observer = ConversationObserver(pool)

      # Watch history file
      event_handler = Observer()
      event_handler.schedule(observer, path=str(HISTORY_FILE.parent), recursive=False)
      event_handler.start()

      print("👁️  L1 Observer started. Monitoring conversations...")

      try:
          while True:
              await asyncio.sleep(1)
      except KeyboardInterrupt:
          event_handler.stop()

      event_handler.join()
      await pool.close()

  if __name__ == "__main__":
      asyncio.run(main())
